## Register

**brand** — design IS the product. This is a corporate marketing site for a Lviv real-estate developer, not an internal SaaS / dashboard / app shell. Apply Impeccable's `reference/brand.md` rules.

## Product Purpose

Корпоративний хаб-сайт львівського забудовника **«ВИГОДА»** (юр. назва: ТОВ «БК Вигода Груп», ЄДРПОУ `42016395`, ліцензія від 27.12.2019).

Поточний deliverable — **клікабельний desktop-first MVP-прототип**, задеплоєний на GitHub Pages, який клієнт отримує як публічний URL для демо. Сайт показує **5 об'єктів портфеля** (0 зданих · 1 активний Lakeview · 4 pipeline) через лінзу «системного девелопменту».

**Критерій успіху:** на 1920×1080 сайт виглядає «дорого» і клікається між 5 ключовими екранами без розвалу.

## Users

Три персони (з `CONTEXT.md` §4):

1. **Інвестор-професіонал** (35–55, капітал $50–200k). Шукає юрособу/ЄДРПОУ, ліцензії, портфель з явними стадіями. Реагує на доказовість і прозорість, не на обіцянки. Страх: «ще один Рубікон». Конверсія — заявка на зустріч / запит презентації.

2. **Покупець бізнес-класу** (30–50, родина, часто переселенець з Києва/Харкова). Шукає реальні фото будівництва, планування, безпеку (укриття, СС3), розтермінування. Конверсія переважно **на сайті ЖК**, корпсайт працює як «фільтр довіри».

3. **Банк / фінансовий партнер** (due diligence). Шукає повні реквізити, ліцензії, дозволи, документи. Конверсія — відкритий B2B-блок.

## Brand Essence

- **Дескриптор:** «системний девелопмент» (фіксований, іде під лого).
- **Гасло-ядро:** *«Системний девелопмент, у якому цінність є результатом точних рішень.»* — канон, не перефразовувати без узгодження.
- **Суть:** девелопер, що продає не квадратні метри, а **цілісне середовище** (архітектура + функціональність + інвестиційна доцільність як одна система).
- **4 цінності:** системність · доцільність · надійність · довгострокова цінність.
- **4 атрибути характеру:** раціональний · впевнений · системний · сучасний.

## Tone of Voice

**Чітко · Впевнено · Предметно · Стримано.**

Без пафосу, без зайвих емоцій, без декоративних слів. Коротко, структуровано, з акцентом на логіку і користь.

**Заборонений словник:** `мрія`, `найкращий`, `унікальний`, `преміальний стиль життя`, `ваша історія починається тут`, `чарівний`, `неповторний`. Жодних маркетингових суперлативів.

**Практичні правила:**
- Заголовки декларативні, короткі: «Середовище, яке залишається актуальним», а не «Розпочніть свою мрію».
- Мікрокопі інструктивне: «Оберіть поверх», «Залиште номер — зателефонуємо сьогодні».
- CTA функціональні: «Переглянути планування», «Отримати розрахунок», «Записатись на огляд». Не «Почніть свою мрію».
- Факти над обіцянками: ЄДРПОУ, ліцензія від 27.12.2019, клас наслідків СС3, конкретна дата здачі.
- Військовий контекст — стримано, без експлуатації болю; укриття подаємо як технічну специфікацію, не як емоційний тригер.

## Anti-references

Чого ВИГОДА **не є** (явні точки розрізнення):

- **Не Avalon / VD Group / Parus Premium** з мовою «преміум-золото», «стиль життя», «емоція». Наша мова — **системність, цифри, доказовість рішень**.
- **Не «20 років досвіду» і «тисячі щасливих мешканців».** Поточна реальність: 0 зданих, 1 активний, 4 у pipeline. Імітувати масштаб — табу. Меседж: «ми ретельно підходимо до кожного рішення, ось як це виглядає на практиці».
- **Не «команда мрії» з фото керівників.** Рішення клієнта: **жодних фото і імен команди**. Довіра — через юридичний фактаж (ЄДРПОУ, ліцензії, дозволи, договори).
- **Не «генеричний AI-лук» з градієнтами, glow, неоном, glassmorphism.** Бренд — каркасна ізометрія + темний фон + точкова лайм-акцент.

## Strategic Principles

1. **Доказовість > обіцянки.** Кожне твердження має бути перевірюваним: цифра, дата, документ, реєстр.
2. **Закрита палітра, закрита типографіка.** 6 кольорів і 1 шрифт (3 ваги) — без винятків. Бренд — це і є дисципліна.
3. **Ізометричний куб як візуальний якір.** Каркасна wireframe-мова присутня всюди — від лого до фонових патернів і UX-станів «pipeline без рендерів».
4. **Silent displacement — лише Lakeview.** Тема Pictorial / Рубікон публічно НЕ ініціюється: ні досьє, ні хронологій, ні контр-наративу. Інші 4 pipeline-проекти мають власні робочі назви — silent displacement до них **не застосовується**.
5. **Фото — лише з `/renders/` і `/construction/`.** Жодних стокових ілюстрацій, жодних AI-генерованих архітектурних візуалів.
6. **Skeptic-pass перед edit-pass.** Будь-яке inherited твердження з CONCEPT/CONTEXT/memory флажиться, якщо немає документального підкріплення; не тихо розширюється.

## Constraints (Hard Limits)

- **Stack:** Vite 6 + React 19 + Tailwind v4 + Motion + react-router-dom 7. GitHub Pages static-only — без CMS, без SSR.
- **Performance:** Lighthouse Desktop ≥ 90 усі 4 категорії; hero render ≤ 200KB (AVIF/WebP); bundle ≤ 200KB gzipped JS.
- **Accessibility floor:** WCAG 2.1 AA для normal text. Текст `#A7AFBC` на `#2F3640` — тільки для ≥14pt (контраст 5.3:1). `#C1F33D` — **ніколи** на світлому фоні (контраст ≈1.2:1, FAIL).
- **Viewport:** desktop-first 1920×1080. Нижче 1280px — graceful fallback без розвалу. Mobile responsive — НЕ v1.
- **Browser support:** last 2 versions Chrome / Safari / Firefox / Edge. Без IE11.
